Справка по системе

IBSec LMS — корпоративная система обязательного обучения по информационной безопасности.

Назначение системы

Система помогает назначать обязательное ИБ-обучение, контролировать сроки, проверять знания, собирать отчётность и вести аудит действий.

Основные разделы
  • Курсы — каталог и материалы обучения.
  • Назначения — планирование обязательного обучения.
  • Тесты — проверка знаний и попытки сдачи.
  • Отчеты — контроль прогресса и выгрузки CSV.
  • Аудит — журнал действий пользователей.
  • Импорт оргструктуры — загрузка сотрудников, подразделений, должностей и ролей из CSV.
Роли пользователей
  • employee — проходит назначенные курсы и тесты, видит личный прогресс.
  • security_officer — назначает обучение, смотрит отчеты, аудит и журнал импорта.
  • admin — управляет системой и имеет доступ к Django admin.
Инструкция для сотрудника
  1. Откройте «Мои назначения».
  2. Перейдите в курс.
  3. Изучите материалы и пройдите тест.
  4. Проверьте результат и личный прогресс.
Инструкция для ответственного за ИБ
  1. Назначьте обязательные курсы сотрудникам.
  2. Проверьте dashboard и отчеты.
  3. Откройте журнал аудита при проверке действий.
  4. Используйте раздел импорта оргструктуры для загрузки данных из CSV.
Инструкция для администратора
  1. Проверяйте роли и доступы пользователей.
  2. Используйте Django admin для справочников и контроля.
  3. Следите за результатами импорта и журналом аудита.
  4. Проверяйте выпуск отчётов и сбор статических файлов.
