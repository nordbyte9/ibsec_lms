# IBSec LMS (Django)

Учебно-корпоративная система обучения и контроля знаний сотрудников по информационной безопасности.
Готова для демонстрации комиссии и вставки скриншотов в ВКР.

## Быстрый старт (локально)

```bash
python -m venv .venv
. .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt

python manage.py migrate
python manage.py createsuperuser  # (по желанию)
python manage.py seed_demo        # создаст демо-данные и пользователей

python manage.py runserver
```

### Демо-доступы
- Администратор: admin@example.com / admin12345
- Преподаватель: instructor@example.com / instructor12345
- Сотрудник: employee@example.com / employee12345

## Что есть в проекте
- Роли пользователей: Администратор, Преподаватель, Сотрудник
- Курсы, уроки (текст, ссылки, файлы)
- Тесты (множественный выбор), автоматическая проверка и результаты
- Отчётность: личный прогресс, аналитика по курсам/тестам
- Уведомления (консольная почта), логирование действий
- Командa `seed_demo` для быстрой демонстрации

## Настройка БД PostgreSQL (опционально)
В `settings.py` оставлена конфигурация SQLite по умолчанию. Для PostgreSQL раскомментируйте блок `DATABASES` и укажите свои параметры.

## Скриншоты для диплома
- Экран авторизации (`/accounts/login/`)
- Список курсов (`/courses/`)
- Детали курса и уроков
- Прохождение теста
- Отчёты (`/reports/my/`, `/reports/analytics/`)
