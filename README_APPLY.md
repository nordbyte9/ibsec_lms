# IBSec LMS — пакет финальной очистки

Пакет рассчитан на актуальную ветку `main` репозитория `nordbyte9/ibsec_lms`.
Удалённые ранее файлы из пункта 1 он не восстанавливает и не изменяет.

## Что исправляется

- удаляются устаревшие маршруты и функции отчётов;
- исправляется отображение процента в разделе «Мой прогресс»;
- удаляются пустые шаблонные проверки и подписи `OK`;
- локализуются действия и типы объектов журнала аудита;
- справка приводится к виду документации готового продукта;
- публичная регистрация отключается;
- исправляется перенаправление после сохранения профиля;
- нейтрально переписываются неподтверждённые заявления на странице входа;
- модуль уведомлений становится рабочим: список, прочтение и автоматическое создание уведомления при назначении курса;
- добавляются тесты уведомлений и удалённых маршрутов.

## Применение

1. Скопируйте `apply_cleanup.py` в корень проекта, рядом с `manage.py`.
2. Убедитесь, что текущие изменения сохранены в Git.
3. Выполните:

```powershell
python apply_cleanup.py
python manage.py migrate
python manage.py check
python manage.py makemigrations --check --dry-run
python manage.py test
```

4. Проверьте изменения:

```powershell
git status
git diff --check
git diff
```

## Рекомендуемый коммит

```powershell
git add -A
git commit -m "Finalize project cleanup and notifications"
git push origin main
```

После успешного применения сам файл `apply_cleanup.py` можно удалить перед коммитом, если вы не хотите хранить служебный сценарий в репозитории.
