# Экспорт документов

IBSec LMS формирует XLSX-реестры обучения и DOCX-сертификаты.

## XLSX-реестр

Реестр доступен пользователям с правами ответственного за ИБ, администратора или суперпользователя. Экспорт учитывает активные фильтры отчёта и включает сведения о сотрудниках, подразделениях, курсах, сроках, статусах и результатах тестирования.

Значения, начинающиеся с символов `=`, `+`, `-` или `@`, экранируются перед записью в Excel. Это предотвращает интерпретацию пользовательских данных как формул.

## DOCX-сертификат

Сертификат формируется только для завершённого назначения курса. Сотрудник может получить собственный документ; ответственный за ИБ, администратор и суперпользователь — документ для доступного им завершённого назначения.

Реквизиты задаются через переменные окружения:

```env
DOCUMENT_ORGANIZATION_NAME=IBSec LMS
DOCUMENT_SIGNER_TITLE=Ответственный за информационную безопасность
DOCUMENT_SIGNER_NAME=Иванов И.И.
```

## Аудит

Формирование XLSX и DOCX фиксируется в журнале аудита с указанием пользователя и типа операции.

## Проверка

```bash
python manage.py test tests.test_document_exports
```

Тесты открывают созданные файлы библиотеками `openpyxl` и `python-docx`, а также проверяют обработку управляющих символов и потенциальных Excel-формул.
