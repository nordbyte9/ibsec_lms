# Smoke Checklist

- start the project and open the homepage;
- log in as `admin`;
- log in as `security_officer`;
- log in as `employee`;
- open the course catalog;
- open the assignments page;
- pass a test with a demo employee;
- verify reports;
- verify the audit journal;
- verify the integrations page;
- verify CSV import with `import_org_csv`;
- verify Django admin access.
